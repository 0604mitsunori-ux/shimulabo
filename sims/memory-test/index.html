あそぶ・実験

記憶力テスト

4つのパネルが光る順番を覚えて、同じ順にタップ。正解するたびに1つ長くなります。あなたは何個まで覚えられる？（短期記憶のトレーニングにも）

人が一度に覚えられるのは7±2個と言われます（マジカルナンバー）。どこまでいける？

マジカルナンバー7

人間が短期記憶で一度に保持できる情報は「7±2個」が目安とされます（ミラーの法則）。電話番号がこのくらいの桁なのも偶然ではありません。塊（チャンク）にまとめると、もっと覚えられるようになります。
